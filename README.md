# User-Registration-Form

A simple user registration Form to learn form elements.

## Form Elements Used:

- Form tag
- input tag
  - text
  - radio
  - select
  - checkbox
  - placeholder
  - required attribute
- fieldset
- submit button
- meta tags

## Psudo Selectors
- :hover
- :focus

## Media Queries
To make the website Mobile Responsive
